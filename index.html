Project Overview

About Project

AP Quiz Application is a desktop-based quiz system developed using Java Swing and Core Java concepts. The application allows users to participate in quizzes while administrators can dynamically manage quiz questions.

Project Objective

The objective of this project is to provide a modern, efficient, and user-friendly platform for conducting digital quizzes with automated score calculation and leaderboard management.

Key Features

Admin Panel

Secure admin authentication with Add, Edit, Delete, and Dynamic Question Management functionality.

Quiz System

Interactive quiz interface with timer, MCQ options, navigation controls, and automatic score calculation.

Leaderboard

Displays user rankings according to quiz scores and improves competitiveness among participants.

Modern GUI

Stylish Java Swing graphical interface with responsive layouts, custom buttons, and modern UI design.

Technologies Used

Java Core Java Java Swing AWT File Handling OOP NetBeans GUI Design

Project Modules
